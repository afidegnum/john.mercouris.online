Overview
========================================================================
This is a pelican theme using `Pure.css <http://purecss.io/>`_.


How to Install?
========================================================================
Clone this repository:

Configure your :code:`pelicanconf.py` as:

.. code-block:: python

   % THEME = /path/to/theme

Favicon
------------------------------------------------------------------------
Set :code:`FAVICON` to specify a site favicon:

.. code-block:: python

   FAVICON = 'images/favicon.ico'
